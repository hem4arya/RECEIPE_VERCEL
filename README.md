# RECEIPE_VERCEL

A web-based recipe application built using JavaScript, CSS, and HTML. This project lets users browse, search, and interact with a collection of recipes, providing a delightful cooking experience.

## Features

- Browse a wide variety of recipes
- Search for recipes by name or ingredient
- Responsive design for desktop and mobile devices
- Clean and intuitive user interface

## Tech Stack

- **JavaScript:** Core application logic and interactivity (87.3%)
- **CSS:** Styling and layout (8.9%)
- **HTML:** Structure and content (3.8%)

## Getting Started

1. **Clone the repository:**
   ```bash
   git clone https://github.com/hem4arya/RECEIPE_VERCEL.git
   cd RECEIPE_VERCEL
   ```

2. **Run locally:**
   - Open `index.html` in your browser.

3. **Deploy:**
   - Easily deploy to [Vercel](https://vercel.com/) or any static hosting provider.
